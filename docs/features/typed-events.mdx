---
title: Type-Safe Events
description: Define validated event payloads and event result types.
---

Events are strongly typed and validated in both runtimes.

Repository example files:
- [`examples/simple.py`](https://github.com/pirate/bbus/blob/main/examples/simple.py)
- [`abxbus-ts/examples/simple.ts`](https://github.com/pirate/bbus/blob/main/abxbus-ts/examples/simple.ts)

<Tabs>
<Tab title="Python">

```python
from typing import Any
from abxbus import BaseEvent

class OrderCreatedEvent(BaseEvent[dict[str, Any]]):
    order_id: str
    customer_id: str
    total_amount: float
```

</Tab>
<Tab title="TypeScript">

```ts
import { BaseEvent } from 'abxbus'
import { z } from 'zod'

const OrderCreatedEvent = BaseEvent.extend('OrderCreatedEvent', {
  order_id: z.string(),
  customer_id: z.string(),
  total_amount: z.number(),
  event_result_type: z.object({ ok: z.boolean() }),
})
```

</Tab>
</Tabs>

- Python payload validation is powered by Pydantic models.
- TypeScript payload and result validation is powered by Zod schemas.
