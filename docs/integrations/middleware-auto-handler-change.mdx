---
title: AutoHandlerChangeEventMiddleware
description: Emit auto events when handlers are registered/unregistered.
---

`AutoHandlerChangeEventMiddleware` emits metadata events whenever `EventBus.on(...)` or `EventBus.off(...)` changes handler registration.

## Constructor params

None.

## Setup with EventBus

<Tabs>
<Tab title="Python">

```python
from abxbus import EventBus
from abxbus.middlewares import AutoHandlerChangeEventMiddleware

bus = EventBus(
    name='AppBus',
    middlewares=[AutoHandlerChangeEventMiddleware()],
)
```
</Tab>
<Tab title="Rust">

```rust
// Not implemented in abxbus-rust yet.
// Use explicit typed handler wrappers around bus.on(...) for this behavior for now.
```

</Tab>
</Tabs>


## Behavior

- On registration, emits `BusHandlerRegisteredEvent(handler=...)`.
- On unregistration, emits `BusHandlerUnregisteredEvent(handler=...)`.
- Emits a deep-copied handler metadata object.
- Useful for auditing dynamic handler topology.
